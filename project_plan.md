# SIBBOARD — Сибирская доска объявлений

## 1. Описание проекта
SIBBOARD — футуристичная доска объявлений для Новосибирска с концепцией "Сибирский хайтек". Сочетает грубую красоту Сибири с ультра-современным UI: 3D-горы, glassmorphism, плавные анимации. Позиционирование: "Не доска объявлений — а цифровой соседский центр."

**Целевая аудитория**: Жители Новосибирска, покупающие и продающие вещи между соседями.

## 2. Структура страниц
- `/` — Главная (Hero 3D + поиск + категории + объявления + районы + CTA)
- `/ads/:id` — Карточка объявления (галерея + история вещи + продавец + карта)
- `/dashboard` — Личный кабинет (статистика + объявления + достижения)

## 3. Ключевые фичи
- [ ] 3D-горы на фоне (Three.js / @react-three/fiber)
- [ ] Glassmorphism UI с тёмной темой
- [ ] AI-ассистент "Сибиряк" (чат-интерфейс)
- [ ] Аудио-визитки продавцов (Web Audio API)
- [ ] "Соседский радар" — карта с продавцами
- [ ] Истории вещей (эмоциональные чипсы)
- [ ] Геймификация (значки достижений)
- [ ] Анимации Framer Motion
- [ ] Тёмная/светлая тема

## 4. Дизайн-система
- Цвета: #0B4F6C (Обь), #E6B31E (мёд), #F7F9FC (снег), #1A3B2E (тайга), #0A1828 (ночь)
- Шрифт: Nunito (Google Fonts)
- Скругления: 32px карточки, 48px кнопки
- Стиль: Glassmorphism + тёмная тема

## 5. Интеграции
- Supabase: планируется (авторизация, БД объявлений)
- Карты: 2ГИС embed
- AI: YandexGPT API (будущая фаза)

## 6. Фазы разработки

### Фаза 1: Главная страница + 3D + UI ✅
- Цель: Создать визуально ошеломляющую главную с 3D-горами
- Результат: Полная главная страница с анимациями

### Фаза 2: Карточка объявления
- Цель: Детальная страница объявления
- Результат: /ads/:id с галереей, историей, продавцом

### Фаза 3: Личный кабинет
- Цель: Дашборд пользователя
- Результат: /dashboard со статистикой и достижениями

### Фаза 4: Интерактивность
- Цель: AI-ассистент, аудио-визитки, карта
- Результат: Полный интерактивный опыт

### Фаза 5: Supabase интеграция
- Цель: Реальные данные и авторизация
- Результат: Рабочий MVP
